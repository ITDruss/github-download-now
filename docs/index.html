GitHub Download Now — Privacy Policy

Effective date: July 13, 2026

GitHub Download Now does not sell user data and does not send data to the developer, analytics providers or advertising services.

GitHub API requests

When release metadata is required, the extension sends the public repository owner and repository name to GitHub's official API at api.github.com. Background requests are made only for repositories the user explicitly chooses to watch.

Local storage

Optional download records, watched repositories, pending updates and ETags are stored locally on the current device. The extension does not read the browser's general download history. Preferences may be synchronized by the browser through storage.sync.

Background checks and notifications

Browser alarms check selected repositories according to the user's chosen interval. Disabling the extension or selecting manual-only checks stops scheduled requests. System notifications are optional.

Data not accessed

The extension does not access cookies, GitHub tokens, account profile data, personal communications, the browser's browsing-history database or unrelated websites.

Limited Use

Information received through browser APIs is used only to provide release-download and update-tracking features. It is not used for advertising, profiling, sale to third parties, creditworthiness or unrelated purposes, and is not made available for human review by the developer.

User controls

Users can disable the extension, choose manual update checks, remove watches, clear local history, disable notifications or uninstall the extension.
