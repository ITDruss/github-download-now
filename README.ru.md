# GitHub Download Now

GitHub Download Now — open-source расширение для Chromium и Firefox, которое помогает выбрать, скачать и отслеживать обновления программ, опубликованных через GitHub Releases.

Расширение добавляет на страницы репозиториев заметную кнопку загрузки, определяет операционную систему и архитектуру процессора, рекомендует наиболее подходящий файл, группирует альтернативы по платформам и при желании следит за новыми релизами выбранных проектов.

![GitHub Download Now](assets/social-preview.png)

## Возможности

- определение Windows, Linux, macOS и Android;
- поддержка x64, x86, ARM64 и ARM;
- выбор подходящих установщиков и пакетов для каждой платформы: EXE и MSI, DMG и PKG, AppImage, DEB и RPM, APK и других форматов;
- структурированное меню альтернативных сборок;
- исключение checksum, подписей, SBOM, debug-файлов и исходников из автоматической рекомендации;
- поиск явной инструкции по сборке по запросу пользователя в документации тега релиза с показом исходных команд и ссылки на файл;
- настройка предпочитаемого формата пакета;
- ручное переопределение ОС и архитектуры;
- предупреждение об устаревших релизах;
- объяснение причин выбора файла;
- необязательная локальная история загрузок через расширение;
- добровольное слежение за обновлениями выбранных репозиториев;
- popup со списком обновлений, badge и необязательные системные уведомления;
- русский и английский интерфейс.

## Приватность

В расширении нет аналитики, рекламы, собственного сервера и доступа к аккаунту GitHub.

Публичные имя владельца и название репозитория отправляются только в официальный GitHub API, когда нужны метаданные релиза. Фоновые запросы выполняются только для репозиториев, за которыми пользователь явно разрешил следить. История загрузок и подписки хранятся локально на текущем устройстве. Настройки могут синхронизироваться средствами самого браузера через `storage.sync`.

Полная политика: [PRIVACY.md](PRIVACY.md).

## Установка для тестирования

### Chromium

1. Распаковать `github-download-now-chromium-v0.4.2.zip`.
2. Открыть `chrome://extensions`.
3. Включить **Режим разработчика**.
4. Нажать **Загрузить распакованное расширение**.
5. Выбрать папку с `manifest.json`.

### Firefox

1. Распаковать `github-download-now-firefox-v0.4.2.zip`.
2. Открыть `about:debugging#/runtime/this-firefox`.
3. Нажать **Загрузить временное дополнение**.
4. Выбрать `manifest.json`.

Для постоянной установки Firefox-сборка должна быть подписана Mozilla через AMO.

## Разработка

Нужны Node.js 20+ и Python 3.

```bash
npm ci
npm test
npm run package
```

## Разрешения

- `storage` — настройки, локальная история и список отслеживаемых репозиториев;
- `alarms` — расписание проверок обновлений;
- `notifications` — необязательные системные уведомления;
- `github.com` — отображение кнопки на странице репозитория;
- `api.github.com` — получение публичных метаданных релизов.

Расширение не запрашивает cookie, историю браузера, GitHub Token и общую историю загрузок.
